SOLAMI: End-to-End Social Multimodal Model

Immersive Interaction with 3D Autonomous Characters

SOLAMI is the first end-to-end social multimodal model framework for immersive interaction with 3D autonomous characters. Through a unified vision-language-action architecture, automatically generated multimodal interaction dataset, and immersive VR interface, it achieves more natural, precise, and low-latency human-machine social interaction, providing a new solution for building socially intelligent 3D autonomous characters.

AnimoXtend

AIGC 3D Generation Plugin

An innovative AIGC 3D generation plugin that brings a new creative experience to Blender users. Through natural language interaction, users can easily generate scenes and character models, and achieve automated character rigging. The plugin includes over 4,000 professional-grade motion capture data and supports flexible retargeting of these animations to any character model. As an all-in-one AI-assisted creation tool, AnimoXtend is dedicated to helping 3D creators efficiently complete the entire workflow from modeling to animation in Blender, significantly improving creation efficiency.

Manned Lunar Landing VR Simulation

Originating from the China Astronaut Research and Training Center, this project fully simulates the manned lunar landing mission process through VR equipment integrated with flight joystick simulation and wearable gesture recognition sensors. The system includes multiple functional modules such as an interactive command cabin control panel, simulated augmented reality virtual interface, and remotely operable lunar rover.
